Window下OCaml和opam安装教程

发表于2021-08-05|更新于2021-08-09|工具教程
|字数总计:281|阅读时长:1分钟|阅读量:

博主本人安装opam主要是为了安装Lustre V6，虽然不知道管不管用，但是先安装了再说。(●’◡’●)

Opam install

使用Graphical Installer应该可以一步到位，而不需要自己装opam和cygwin，下载Graphical Installer下的64-bit得到OCaml64.exe。

  1. 获取安装包

    下载64位：https://fdopen.github.io/opam-repository-mingw/installation/

  2. 通过Graphical Installer安装OCaml

    打开OCaml64.exe，除了改下安装位置，Next就可以了

    第一次遇到问题，在刚才的cygwin里面至少有一个tool安装失败了。

    删掉重新安装吧。T^T

    然后开始了漫长的无数次重装，可能是网的问题，换个地方就好了，也可能是风水的问题。

    安装成功。

  3. 测试是否安装成功

    打开cygwin64终端，查看opam和OCaml版本。

  4. 安装depext

    应该是个类似于pip的东西。

  5. 简单使用

    ocaml输出hello world。

    暂时没有找到添加路径的方法。
